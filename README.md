# 📁 Projetos

## 1️⃣ Dashboard de Faturamento Clínico (MyBI)

### 📌 Contexto
Diretoria da clínica precisava acompanhar indicadores operacionais e gerenciais para garantir eficiência no atendimento e sustentabilidade do negócio.

Este dashboard foi desenvolvido para fornecer **uma visão consolidada da performance clínica**, permitindo que gestores identifiquem gargalos e oportunidades de melhoria.

### 🎯 Objetivo
Monitorar indicadores estratégicos relacionados a:

- Volume de atendimentos
- Faturamento hospitalar
- Distribuição de pacientes por atendimento
- Performance de especialidades médicas

### 📊 Principais KPIs

- Valor total faturado
- Valores entregues para faturamento
- Atendimentos por unidade
- Ranking de atendimentos por convênio e especialidade
- Ranking de serviços mais realizados

### 🛠 Ferramentas Utilizadas

- SQL SERVER
- MyBI
- Modelagem de dados

### 🔎 Principais Insights

- Identificação de especialidades com maior volume de pacientes
- Tendência de crescimento ou queda nos atendimentos
- Avaliação das operadoras de saúde com maior taxa de serviços atendidos

## 2️⃣ Dashboard de Análise Financeira Operacional (MyBI)

### 📌 Contexto
Cliente enfrentava dificuldades para identificar as pendências de recebimentos e não existia controle quanto à quitações, além de ter a necessidade de manter uma previsão de recebimentos.

Este dashboard permite acompanhar a previsão dos recebimentos e pendências

### 🎯 Objetivo
Analisar os títulos a receber e recebidos

### 📊 Principais KPIs

- Total de títulos A Receber, Recebidos, Pendências e Glosas
- Valores previstos a receber x valores recebidos
- Histórico de lançamentos
- Controle de quitações
- Pendências por pessoas

### 🛠 Ferramentas Utilizadas

- SQL Server
- MyBI

### 🔎 Principais Insights

- Identificação dos produtos com maior contribuição no faturamento
- Regiões com melhor desempenho comercial
- Tendências sazonais nas vendas
